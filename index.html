Where did javascript come from?

document.write("Hello Open Source Bridge!");
  • So Where did Javascript come from?
  • Originally, it was developed as a server-side scripting language
  • Developed by Brendan Eich in 1995, while he was working for Netscape.
  • Supposedly he developed it in a week
  • Which adds to its cred and also it criticism
  • it was developed under the name Mocha, it shipped being called Livescript, and was later changed to JavaScript aka ECMAScript.
  • They attached the name Java to confuse and hopefully attract developers that were using the hot new langauage Java.
  • Begining in 1996 standards were written, Netscape pushed it into the Browser and of course Microsoft refused to play along and developed JScript a port of javascript.
  • which contributed to what is known as the browser wars
  • this devide left javascript and jscript both neglected and it wasn't until
  • 1996, Netscape announced that it had submitted JavaScript to Ecma International for consideration as an industry standard, and subsequent work resulted in the standardized version named ECMAScript.
  • Currently javascript is at ECMAScript 6 published in 2015

This DOM?

So what this Dom we talking about? Its not this guy?

Not some new slage for these guys?

Nah, This DOM!

We are talking about The Document Object Model or the "DOM" is a cross-platform and language independent convention for representing and interacting with objects in HTML

With the HTML DOM, JavaScript can access and change all the elements of an HTML document

  • The history of the Document Object Model is intertwined with the history of these "browser wars"
  • eventually W3C began working on a standardized DOM
    • DOM Level 2 was published in late 2000. It introduced the "getElementById"
    • the current release, DOM Level 3 came out in 2003
    • and DOM Level 4 is being developed

What was javascript intended for?

  • Javascript was intended to be a language that would complement Java by appealing to nonprofessional programmers
  • it was designed to run in Netscape Navigator and it's success in the early broswers led to it becomeing the standard scripting language of the web
  • Which is why you can find it in every modern browser today.
  • Only within the last 10 years have people really started using it as a dynamic object-oriented "programming" language
