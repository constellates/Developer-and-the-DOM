Where did javascript come from?

  • Originally a server-side scripting language, JavaScript was developed by Brendan Eich in 1995, while he was working for Netscape.
  • He developed it in a week (so he says).
  • Developed as Mocha, shipped as Livescript, and later changed to JavaScript (but also known as ECMAScript) to latch onto the populararity Java had.
  • Begining in 1996 standards were written, Netscape pushed it into the Browser and Microsoft refused to play along.
  • And the DOM was born.
  • To render a document such as an HTML page, most web browsers use an internal model similar to the DOM.
  • The limited facilities for detecting user-generated events and modifying the HTML document in the first generation of these languages eventually became known as "DOM Level 0" or "Legacy DOM.
  • eventually W3C began working on a standardized DOM
    • DOM Level 2 was published in late 2000. It introduced the "getElementById" f
    • the current release DOM Level 3 came out in 2003
    • and DOM Level 4 is being developed

What was javascript intended for?

  • Javascript was intended to be a language that would complement Java by appealing to nonprofessional programmers
  • funny how it is still treated as a lesser language when compared with more traditional langauges.
  • Javascript was designed to run in Netscape. It's success led to it becomeing the standard scripting language of the web
  • Which is why you can find it in every modern browser today.
  • And only within the last 10 years have people really started using it as a dynamic object-oriented "programming" language
